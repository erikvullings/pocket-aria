# What is a "Song" in PocketAria?

## Overview

In PocketAria, a **Song** is the main unit of organization. It represents a single vocal piece in your repertoire and can contain multiple types of content.

## What a Song Can Include

A Song can contain any combination of the following:

### 1. üìù **Metadata** (Required)
- **Title** (required)
- Composer
- Genre (classical, pop, jazz, choir, folk, other)
- Voice Type (soprano, mezzo, alto, tenor, baritone, bass, other)
- Year (when it was written/released)
- Tags (for categorization)
- Description/notes

### 2. üéµ **Audio Track** (Optional)
- Upload an audio file (MP3, WAV, M4A)
- Could be:
  - A recording of you performing it
  - A professional recording for reference
  - An accompaniment track
  - A practice recording

### 3. üéº **Musical Scores** (Optional, Multiple)
You can add multiple scores per song:
- **PDF scores** - Upload sheet music as PDF
- **MusicXML files** - Upload digital music notation
- **Images** - Upload photos/scans of sheet music (PNG, JPG)

### 4. üìÑ **Lyrics** (Optional)
- Enter lyrics in plain text, Markdown, or HTML format
- View them alongside the audio and score
- Useful for memorization and reference

### 5. üéØ **Cue Points** (Optional)
- Link specific measures to timestamps in the audio
- Helps synchronize the score with the recording
- Useful for practice and reference

## Creating a New Song

1. Click **"New Song"** in the Library
2. Fill in at least the **Title** (required)
3. Add metadata (composer, genre, voice type, etc.)
4. Upload files:
   - **Audio**: Click "Upload Audio" ‚Üí Select MP3/WAV/M4A file
   - **Scores**: Click "Upload Score" ‚Üí Select PDF/MusicXML/Image
   - **Lyrics**: Type or paste into the Lyrics field
5. Click **"Save Song"**

## Managing Your Songs

- **View**: See all content (audio player, scores, lyrics)
- **Edit**: Modify metadata, add/remove files
- **Delete**: Remove from your library
- **Export**: Share via permalink or download as JSON
- **Search**: Find songs by title, composer, tags, etc.
- **Organize**: Add songs to playlists
