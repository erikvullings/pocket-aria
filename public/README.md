# Public Assets

This directory contains static assets for the PocketAria PWA.

## Required PWA Icons

You need to add the following icon files:

- `pwa-192x192.png` - 192x192 icon
- `pwa-512x512.png` - 512x512 icon
- `favicon.ico` - Favicon

You can generate these icons using online tools like:
- https://realfavicongenerator.net/
- https://www.pwabuilder.com/imageGenerator

For now, the app will work without these icons but you'll see warnings in the console.
