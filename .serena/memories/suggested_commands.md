# Suggested Commands

## Development Commands

### Start Development Server

```bash
pnpm run dev
# or
pnpm start
```

Starts `Vite` development server at `http://localhost:5173`

### Build for Production

```bash
pnpm run build
```

Compiles TypeScript and creates production build in `dist/` folder

### Preview Production Build

```bash
pnpm run preview
```

Serves the production build locally for testing

### Type Check

```bash
pnpm run type-check
```

Runs TypeScript compiler in check mode (no emit)

## Package Management

```bash
pnpm install         # Install dependencies
pnpm add <package>   # Add a dependency
pnpm remove <package> # Remove a dependency
```

## Git Commands (Standard)

```bash
git status
git add .
git commit -m "message"
git push
git pull
```

## System Commands (macOS/Darwin)

Standard Unix commands work on Darwin:

- `ls` - list files
- `cd` - change directory
- `grep` - search text
- `find` - find files
- `cat` - display file contents
